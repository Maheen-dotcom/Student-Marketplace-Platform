Project Overview

Student Marketplace Platform is a web-based application designed specifically for students who want to buy and sell educational resources. The platform provides a simple and user-friendly environment where students can share academic materials and connect with other students.

Problem Statement

Students often face difficulties finding affordable books, calculators, laptops, notes and other study materials. Existing marketplaces are generally designed for all users and do not focus on academic needs. This platform solves that problem by creating a dedicated marketplace for students.

Project Objectives

  • Provide a platform for buying and selling academic products.
  • Allow students to manage product listings easily.
  • Enable product search and category filtering.
  • Maintain user and product records in a database.
  • Display dashboard statistics for better management.
  • Demonstrate full-stack web development using FastAPI and PostgreSQL.

Key Features

  • User Registration
  • User Login
  • Add Product
  • View Products
  • Update Product
  • Delete Product
  • Product Search
  • Product Filtering
  • Dashboard Analytics

Technology Stack

Frontend: HTML, CSS, JavaScript

Backend: FastAPI (Python)

Database: PostgreSQL (Supabase)

API Testing: Swagger UI

Version Control: GitHub

Deployment: Vercel & Render

Future Enhancements

  • Product Image Uploads
  • Wishlist Feature
  • Chat Between Buyers and Sellers
  • Ratings and Reviews
  • JWT Authentication
  • Advanced Search Filters
